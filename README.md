# - Algorithms Playground -

> Learn Computer Science by building algorithms from scratch.

Algorithms Playground is my long-term learning project where I implement classic computer science algorithms using Java.

The goal isn't just writing code.

It's understanding how algorithms work, analyzing their complexity, writing clean implementations, and documenting what I learn throughout my Artificial Intelligence Engineering journey.

---

## Why this project?

Algorithms are one of the foundations of Computer Science.

This repository helps me strengthen problem-solving skills, improve my software engineering practices, and build a solid foundation for AI and software development.

Each algorithm includes:

- Clean Java implementation
- Documentation
- Unit tests
- Time & Space complexity analysis
- Practical use cases
