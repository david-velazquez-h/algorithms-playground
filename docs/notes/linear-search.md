# Linear Search

## What is Linear Search?

Linear Search is one of the simplest searching algorithms.

It works by checking each element of an array one by one until the target value is found or the array ends.

---

## Algorithm

1. Start from the first element.
2. Compare it with the target.
3. If it matches, return the index.
4. Otherwise continue.
5. If the array ends, return -1.

---

## Time Complexity

| Case | Complexity |
|------|------------|
| Best | O(1) |
| Average | O(n) |
| Worst | O(n) |

---

## Space Complexity

O(1)

The algorithm uses a constant amount of extra memory.

---

## Advantages

- Easy to understand.
- Very simple to implement.
- Works on unsorted arrays.

---

## Disadvantages

- Slow for large datasets.
- Not suitable for frequent searches.

---

## Real-world examples

- Searching for a contact in a small list.
- Looking for an item in an unsorted collection.
- Finding a specific value in sensor data.
